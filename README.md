# Command

## Commands

### Install dependencies

```sh
flutter pub get
```

### Run application

```sh
flutter emulators --launch <emulator_name>
flutter run
```

### Generate MOOR

```sh
flutter packages pub run build_runner build
```

### Build

```sh
flutter build apk
```

### Run tests

```sh
flutter test
```
